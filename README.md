# BookMyStay – Room Inventory Setup & Management

## Overview

This module manages the **hotel room inventory** for the BookMyStay application.
It provides a centralized system for storing, updating, and retrieving room availability information.

The goal is to maintain a **single source of truth** for hotel room inventory while ensuring fast lookups and consistent data management.

---

# Example Operations

### Setup Default Inventory

```
Single → 10 rooms → ₹1000
Double → 6 rooms → ₹1800
Suite → 2 rooms → ₹3500
```

### Add New Room Type

```
Deluxe → 8 rooms → ₹2500
```

### Update Room Type

```
Single → 12 rooms → ₹1100
```

### Check Availability

```
Single → Available
Suite → Available
Deluxe → Available
```
